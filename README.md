# **Doctors Appointment Platform 🩺**  
[**Live Demo**](https://healthcare-project-sepia.vercel.app/)
 

## Overview  

This project is a **Doctors Appointment Booking Platform** designed for patients to book appointments with doctors specializing in various fields such as dentistry, cardiology, and more. The platform provides a seamless user experience for:  

- **Patients**: Booking and managing appointments, secure payments.  
- **Doctors**: Scheduling, managing, and canceling appointments.  
- **Admins**: Monitoring platform activity, managing users, and overseeing appointments.  

---

## Features 🚀  

### Patient Features  
1. **Search Doctors**  
   - Find doctors based on specialization and availability.  

2. **Book Appointments**  
   - Schedule appointments and make secure payments via Stripe.  

3. **Manage Bookings**  
   - View, reschedule, or cancel appointments.  

### Doctor Features  
1. **Schedule Management**  
   - Add, edit, or delete appointment slots.  

2. **Patient Communication**  
   - Contact patients for updates or follow-ups.  

### Admin Features  
1. **Dashboard Overview**  
   - Monitor site usage, appointments, and transactions.  

2. **User Management**  
   - Add or remove doctors and patients.  

3. **Reports**  
   - Generate appointment and revenue statistics.  

---

## Technologies Used 💻  

- **Framework**: Next.js as the full-stack framework (in TypeScript).  
- **Styling**: Tailwind CSS for modern and responsive design.  
- **Database**: MongoDB for scalable and reliable data storage.  
- **Authentication**: NextAuth.js for secure login with Credentials and Google OAuth.  
- **Payments**: Stripe for handling payments securely.  
- **UI/UX**: Headless UI and Heroicons for components.  
- **Mailing System**: Nodemailer for sending appointment confirmations and reminders.  

---

## How It Works 🛠  

### For Patients  
1. **Sign Up or Login**  
   - Create an account or log in using email or Google.  

2. **Search Doctors**  
   - Browse doctors based on specialties.  

3. **Book Appointments**  
   - Select a time slot, provide details, and pay securely.  

---

### For Doctors  
1. **Sign Up or Login**  
   - Create an account or log in to access the doctor dashboard.  

2. **Manage Schedule**  
   - Add or remove available time slots for appointments.  

3. **Manage Patients**  
   - View upcoming appointments and communicate with patients.  

---

### For Admins  
1. **Dashboard**  
   - Monitor all activities including appointments, payments, and users.  

2. **Manage Users**  
   - Approve or revoke access for doctors and patients.  

3. **Generate Reports**  
   - View site analytics and generate financial reports.
